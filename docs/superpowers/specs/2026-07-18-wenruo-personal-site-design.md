# 温若的个人小站 — 主页设计文档

## 1. 项目概述

「温若的个人小站」是一个以 OC（原创角色）"温若"为核心的个人作品集 + 创意空间网站。温若是一位女性剑修、散修，气质清冷、剑心通明。网站本身被定位为"实实在在的艺术"，使用最前沿的前端技术构建，追求极致优雅的动态体验。

## 2. 设计目标

- 营造水墨仙气、清冷诗意的视觉氛围
- 通过 WebGL 与 GSAP 实现沉浸式的长卷叙事体验
- 每个滚动章节都有独立且连贯的动效转场
- 主页面作为整个站点的入口，先聚焦人物意境，后续再扩展作品、法器、功法等内容

## 3. 站点定位

- **核心定位**：个人作品集 + 创意空间 / 数字花园
- **角色设定**：温若，女性散修，剑修
- **气质关键词**：清冷、诗意、水墨、仙气、剑心通明

## 4. 技术栈

| 层级 | 技术 |
|------|------|
| 3D / WebGL | Three.js（模块化导入） |
| 动画 | GSAP + ScrollTrigger |
| 样式 | 原生 CSS、CSS 变量、@font-face |
| 结构 | 语义化 HTML5 |
| 逻辑 | 原生 ES 模块 |
| 数据 | `data/content.json` 集中管理文案 |

## 5. 页面结构（纵向长卷单页）

### 5.1 开场 · 无名之境

- 全屏 WebGL 水墨粒子海洋
- 初始 1 秒静默，仅显示远山轮廓
- 墨色粒子从屏幕边缘向中心聚拢，聚成「温若」二字
- 「温若」二字按笔画顺序通过 SVG `stroke-dashoffset` 书写绘制，约 2.5 秒
- 副标题「散修 · 剑修」淡入
- 引句「一缕清风，半卷残云。」随后浮现

### 5.2 人物 · 独行

- 左侧卷轴容器从宽度 0% 展开至 100%
- 卷轴内文字通过 mask 逐步显现
- 内容：
  - 出身：散修，无门无派
  - 境界：筑基后期
  - 性格：清冷寡言，剑心通明

### 5.3 尾章 · 留白

- 淡入山水背景
- 结语「道未成，行不止。」
- 提示文案：更多画卷、法宝、功法，待续

### 6.2 字体

- 标题：思源宋体 / 演示悠然小楷
- 正文：思源宋体
- 英文/道号：Cinzel 或 Noto Serif

## 7. 动效设计

| 效果 | 实现方式 |
|------|----------|
| 水墨粒子聚拢 | Three.js Points + 自定义 Shader |
| 毛笔书写动画 | SVG 路径 + stroke-dashoffset + GSAP |
| 云雾纵深 | Three.js 多层透明平面，随滚动上升 |
| 卷轴展开 | GSAP 动画控制容器宽度与文字 mask |
| 鼠标灵气 | Canvas 2D 粒子跟随光标 |
| 章节转场 | 墨滴扩散圆形 mask 过渡 |

## 9. 性能策略

- Three.js 按需要素化导入，避免全量引入
- 移动端降级为 Canvas 2D 简化粒子效果
- 使用 IntersectionObserver 控制非可视区域动画暂停
- 图片资源使用 WebP / AVIF 格式
- 字体使用 `font-display: swap`
